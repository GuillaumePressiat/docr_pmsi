1/// ## pmsi-clio CLI ; en ligne de commande
2///
3/// ### Paramètres par défaut
4/// ```bash
5/// ./target/release/pmsi-clio
6/// ```
7///
8/// ### lire les RSA 2012
9///
10/// Découper la partie fixe des RSA 2012 et les écrire dans un fichier parquet
11///
12/// ```bash
13/// ./target/release/pmsi-clio  \
14///                     -a 12 -e rsa -f ~/Documents/data/mco/290000017.2012.12.rsa \
15///                     -o ~/Documents/data/parquet/rust/rsa/290000017.2012.12.rsa.parquet
16/// ```
17///
18/// ### Accéder à l'aide
19/// ```bash
20/// ./target/release/pmsi-clio -h
21/// ```
22///
23///
24///
25/// ```bash
26/// Requêteur rust PMSI : pour tester rust
27///
28/// Usage: pmsi-clio [OPTIONS]
29///
30/// Options:
31///   -f, --file <file>             |-- un fichier .* PMSi
32///   -a, --annee <annee>           |-- Année du fichier PMSI (séquentielle = format du rsa), 22 par défaut
33///   -e, --extension <extension>   |-- Extension * du fichier PMSI, rsa par défaut [rsa, ano, med, rss.ini.txt, ano.txt, med.txt]
34///   -o, --output <output>         |---- Chemin de sortie d'un fichier avec extension csv ou parquet
35///       --ccam <pattern_ccam>     |------ requêteur rsa/rss : Pattern regex CCAM
36///       --cim <pattern_cim>       |------ requêteur rsa/rss : Pattern regex CIM
37///       --ghm <pattern_ghm>       |------ requêteur rsa/rss : Pattern regex GHM
38///       --grep_rss <pattern_rss>  |------ requêteur rsa/rss : Pattern regex ligne entière du RSA ou du RSS
39///   -h, --help                    Print help
40///   -V, --version                 Print version
41/// ```
42///
43///

45/// ## Exemple d'utilisation plus complet
